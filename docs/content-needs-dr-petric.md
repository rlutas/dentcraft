## 1. VIDEO TESTIMONIALE PACIENTI (Prioritate CRITICA)

Pe homepage exista o sectiune dedicata cu 3 video-uri verticale (stil Reels/Shorts). Fara video-uri reale, aceasta sectiune ramane cu placeholder-uri.

### Ce avem nevoie:

- [ ] **Minim 3 video-uri** cu pacienti multumiti (ideal 5-6)
- [ ] Fiecare pacient trebuie sa spuna pe scurt experienta lor

### Specificatii tehnice:

| Parametru | Valoare |
|-----------|---------|
| **Orientare** | **Vertical (9:16)** - prioritar; se accepta si landscape (16:9) |
| **Rezolutie** | Minim 1080x1920 (Full HD vertical) |
| **Format** | MP4 (H.264) |
| **Durata** | 30-60 secunde per video |
| **Audio** | Clar, fara zgomot de fundal (microfon lavaliera ideal) |
| **Lumina** | Buna, fata pacientului sa fie bine luminata |

### Ce sa contina fiecare video:

1. Numele pacientului (prenume + initiala)
2. Ce tratament a facut (ex: implant, fatete, albire)
3. Cum a fost experienta la clinica
4. Rezultatul - cat de multumit este

### Informatii necesare per video:

```
- Numele pacientului (pentru afisare pe site)
- Tipul tratamentului efectuat
- Rating (1-5 stele)
- Consimtamant semnat pentru publicare online
```

### Varianta alternativa:

Daca nu exista video-uri proprii, se pot folosi si link-uri YouTube/YouTube Shorts. Site-ul suporta embed de la YouTube si Vimeo.

## 2. FOTOGRAFII BEFORE/AFTER (Prioritate CRITICA)

Site-ul are o galerie interactiva cu slider before/after. Pe homepage apare un preview, iar pagina `/galerie` afiseaza toate cazurile. In acest moment, galeria este goala.

### Ce avem nevoie:

- [ ] **Minim 3-4 seturi de calitate** pentru lansare (ideal 8-10)
- [ ] Fiecare set = 1 poza BEFORE + 1 poza AFTER

### Specificatii tehnice:

| Parametru | Valoare |
|-----------|---------|
| **Format** | JPG sau PNG |
| **Rezolutie minima** | 800 x 600 px (ideal 1200 x 900 px) |
| **Aspect ratio** | 4:3 (landscape) - recomandat |
| **Dimensiune max fisier** | 400 KB per imagine (se optimizeaza automat pe site) |

### Reguli IMPORTANTE pentru fotografiere:

1. **Acelasi unghi** - poza before si after trebuie facute din exact acelasi unghi
2. **Aceeasi lumina** - iluminare consistenta in ambele poze
3. **Acelasi fundal** - fundal neutru, identic
4. **Aceeasi distanta** - camera la aceeasi distanta de subiect
5. **Focus pe zona tratata** - dintii sa fie clar vizibili

### Informatii necesare per caz:

```
1. Poza BEFORE
2. Poza AFTER
3. Titlu caz (ex: "Fatete ceramice - zambet complet")
4. Tip tratament / Serviciul asociat
5. Durata tratament (ex: "2 saptamani", "3 luni")
6. Descriere scurta a cazului (2-3 propozitii)
7. Doctorul care a efectuat tratamentul
8. Consimtamant pacient pentru publicare (OBLIGATORIU)
```

### Tipuri de cazuri recomandate (minim 1 din fiecare):

- [ ] Fatete / Estetica dentara
- [ ] Implant dentar
- [ ] Albire profesionala
- [ ] Protetica (coroane, punti)
