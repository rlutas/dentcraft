## Icon Location

All icons are stored in:
```
/public/icons/
```

## Icon Format

- All icons are in SVG format
- Created with Adobe Illustrator 19.0.0
- Viewbox dimensions vary (typically 480x480)
- Single color (black) line art style
- Scalable and suitable for any size display
